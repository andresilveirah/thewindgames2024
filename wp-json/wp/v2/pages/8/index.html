Add block patterns<\/h3>\n\n\n\n

Block patterns are pre-designed groups of blocks. To add one, select the Add Block button [+] in the toolbar at the top of the editor. Switch to the Patterns tab underneath the search bar, and choose a pattern.<\/p>\n<\/div>\n\n\n\n

\n

Frame your images<\/h3>\n\n\n\n

Twenty Twenty-One includes stylish borders for your content. With an Image block selected, open the \"Styles\" panel within the Editor sidebar. Select the \"Frame\" block style to activate it.<\/p>\n<\/div>\n\n\n\n

\n

Overlap columns<\/h3>\n\n\n\n

Twenty Twenty-One also includes an overlap style for column blocks. With a Columns block selected, open the \"Styles\" panel within the Editor sidebar.
